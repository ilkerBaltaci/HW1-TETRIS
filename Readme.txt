CSE 241/501 Object Oriented Programming HW#1 - Tetris Best Fit

- Building project
You should install GNU Make and g++ compiler on your computer.
Run command 'make' in the project directory.
'bin' and 'build-Makefile' folders will be created. 'bin' file contains executable of the project. 
'build-Makefile' file contains object files of the projects.
To run the executable, You must run the commands 'cd bin/' and './HW1-TETRIS'.
